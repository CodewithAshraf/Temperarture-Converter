# Temperature Converter

A simple and user-friendly **Temperature Converter** that allows users to convert values between **Celsius** and **Fahrenheit**.

## Features

- Instantly converts temperatures between Celsius and Fahrenheit.
- Accurate calculations using JavaScript.
- Smooth and interactive user interface.
- Easy to use for anyone.

## How to Use

1. Enter a temperature value in the input field.
2. Select the conversion unit (Celsius or Fahrenheit).
3. See the result instantly displayed.
